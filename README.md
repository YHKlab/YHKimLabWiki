### 초기 설정 안내
#### **mkdocs 관련 configuration 설정:**

pip install -r requirements.txt/

mkdocs build

mkdocs serve

#### **원격 저장소에 연결:**

git clone https://github.com/yhkimlab/YHKimLabWiki.git

git remote -v

#### **수정 사항 업데이트:**

git add .

git commit -m "update tutorial"

git push origin main
